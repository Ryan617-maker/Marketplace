# Projeto - Relatório Automático de Testes

## Como usar

1. Instale as dependências:
   pip install -r requirements.txt

2. Execute o script:
   python gerar_relatorio.py

3. O arquivo será gerado:
   relatorio_testes.docx

## O que o projeto faz

- Executa testes com pytest
- Captura os resultados em JSON
- Gera um relatório profissional em Word automaticamente
